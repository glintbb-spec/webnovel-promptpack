# 프롬프트팩 목차 v2.5 통합본 — LOCK v1.2.1·14파일 표준 버전 정합 카드 매핑

이 문서는 사용자가 14개 파일을 어떤 순서로, 어떤 자료를 어디에서 어디로 넘겨가며 실행해야 하는지 정의한다.

14파일 표준 버전 (V2.5):
- LOCK v1.2.1
- 1번·8번·9번·10번·11번 v2.5 통합본
- 2~7번 v2.1 (V2.5에서 변경 없음)
- orchestrator v2.5 통합본
- 목차 v2.5 (본 문서)

목차의 역할은 세 가지다.

```text
1. 모듈 간 카드 매핑 표 — 어디서 어디로 무엇이 넘어가는가
2. 상태 파일 10종 매핑 — 누가 무엇을 갱신하는가
3. AUTO/STEP 모드 흐름 — 사용자 호출 의무 시점 (V2.5: 100화 약 14~15회)
```
