# 사용자 입력 정리 인계 v2.5

## §A. 다음 단계 전달 카드 6종

각 카드는 어느 단계가 받을지 라벨을 명시한다. 다음 단계가 받으면 자기 카드만 사용하고, 나머지는 그대로 다음 단계로 승계한다.

### §A-1. 2번 트렌드 조사 프롬프트용 카드 (→ 2번 직접 사용)

```markdown
- 장르: 현대판타지 / 작가물 / 회귀물
- 플랫폼: 문피아 감각 + 카카오페이지 남성향 감각
- 목표 독자: 20~40대 남성 웹소설 독자
- 목표 회차: 사용자 입력 없음 — 사용자 승인 필요
- 회차 길이 기준: 카카오페이지 남성향 감각 기준 5,500~6,000자 ±10% / 문피아 감각 병행 시 5,000~6,000자 범위에서 2단계 이후 확정 필요
- 키워드:
  · 회귀한 웹소설 편집자
  · 망한 작가 인생 재시작
  · 전생에 실패한 작품 재연재/개작
  · 독자 반응을 다시 배우는 작가
  · 생활비·마감·조회수·선작 수 압박
  · 경쟁 작가의 소재 선점 시도
  · 담당 편집자와의 신뢰 회복
  · 첫 고정 독자/댓글러
  · 금지 키워드: 표절 미화, 독자 기만, 실명 저격, 혐오 표현, 선정적 착취
- 트렌드 사용 전략: 조사 후 결정 — 2·3단계에서 확정 필요
- 피하고 싶은 것 (사용자 회피):
  · 회귀 지식만으로 모든 문제가 쉽게 풀리는 전개
  · 주인공이 경쟁자를 비열하게 짓밟는 전개
  · 댓글/조회수 묘사만 반복되는 전개
  · 작중작 설명이 길어 본편 진행이 멈추는 전개
  · 지나치게 설명적인 업계 강의체
  · 과도한 메타 발언·밈 위주 개그
  · 감정 과잉 신파
  · 딱딱한 보고서식 문장
- 어휘 고정 / 금지 어휘:
  · 고정 제목 후보: 《이번 생은 독자를 믿는다》, 《망한 작가가 회귀했다》, 《편집자였던 내가 다시 쓴다》
  · 금지 어휘: 실제 작가·플랫폼을 특정하게 보이는 실명 저격성 표현 금지
  · 추가 고정/금지 어휘는 사용자 입력 없음 — 사용자 승인 필요
- 조사에 필요한 추가 조건:
  · 실제 플랫폼/랭킹/성공작 흐름은 2단계에서 조사한다.
  · 이 단계에서는 “현재/요즘/트렌드상” 단정 금지.
  · 작가물·회귀물·플랫폼 연재 경쟁 구조의 상업성 검증 필요.
  · 문피아 남성향과 카카오페이지 남성향 감각이 충돌할 경우, 2단계에서 플랫폼 우선순위 확인 필요.
- 비교 기준 메타 일부 (양산형 식별 보조):
  · 후킹 방식: 회차 초반 300자 안에 생활비·마감·조회수·경쟁 선점 중 하나의 압박을 작동시키고, 회차 말미 5문단 안에 다음 선택의 대가 또는 미해결 반응을 남긴다.
  · 싫어하는 이유: 회귀 지식으로 쉽게 해결하거나, 표절·기만을 성공 전략처럼 미화하거나, 댓글/조회수 숫자만 반복해 본편 갈등이 정지되는 전개는 회피한다.
```

### §A-2. 4번 상업 콘셉트 프롬프트용 카드 (→ 4번에서 사용, 2·3번 거쳐 승계)

```markdown
- 장르: 현대판타지 / 작가물 / 회귀물
- 플랫폼: 문피아 감각 + 카카오페이지 남성향 감각
- 목표 독자: 20~40대 남성 웹소설 독자
- 목표 회차: 사용자 입력 없음 — 사용자 승인 필요
- 키워드:
  · 회귀한 웹소설 편집자 출신 실패 작가
  · 전생에 망친 작품을 이번 생에 대박작으로 고쳐 쓰기
  · 독자를 얕본 오만함의 대가와 회복
  · 플랫폼 알고리즘, 초반 성적, 실시간 댓글 반응
  · 경쟁 작가의 선점 시도와 정면승부
  · 담당 편집자와의 불신→인정 관계
  · 첫 고정 독자/댓글러를 통해 독자를 사람으로 인식하는 변화
- 차별화 방향: 1번에서는 최종 확정하지 않음. 사용자 의지값은 “회귀 지식 치트가 아니라 독자를 존중하는 재학습과 작품 구조 개선으로 성공한다”이며, 4번에서 트렌드 조사 결과와 통합해 도출한다.
- 절대 양보 못하는 한 끗:
  · 주인공은 표절하지 않는다.
  · 독자를 속이지 않는다.
  · 후배 작가를 짓밟지 않는다.
  · 성공의 쾌감은 꼼수·표절이 아니라 독자가 진짜 원하는 재미를 정확히 짚어 정면승부로 이기는 데서 나온다.
- 피하고 싶은 것 (사용자 회피): §A-1과 동일.
- 어휘 고정 / 금지 어휘: §A-1과 동일.
- 사용자 톤 단서:
  · “독자를 얕본 오만함”
  · “독자가 진심으로 기다리고 좋아하는 이야기를 쓰는 작가”
  · “악플과 낮은 조회수를 실력으로 뒤집는 순간”
  · “꼼수나 표절이 아니라 독자가 진짜 원하는 재미”
  · “독자를 숫자가 아니라 기다리고 반응하는 사람”
  · “매번 선택과 대가가 있어야 한다”
- 비교 기준 메타 카드 (전체):
  · 좋아하는 이유: 실패한 작가가 자기혐오에서 다시 쓰는 용기로 이동하고, 독자를 숫자가 아니라 사람으로 받아들이는 변화가 상업적 통쾌함과 감정선을 동시에 만든다.
  · 싫어하는 이유: 표절 미화·기만·쉬운 회귀 치트·업계 강의체·댓글 숫자 반복은 작품을 납작하게 만들고 사용자 의지와 충돌한다.
  · 후킹 방식: 회차 초반 300자 안에 압박 1개, 회차 말미 5문단 안에 선택의 대가 또는 미해결 반응 1개.
  · 대사 밀도: 대사 45~55%, 서술 45~55%. 편집자·경쟁자·댓글 반응 장면은 대사/짧은 반응을 늘리고, 작중작 설명은 1회 3문단 이내로 제한한다.
  · 장면 전환 속도: 회차당 4~5개 씬. 각 씬은 “압박 제시→판단→선택/반응” 중 최소 2단계를 포함한다.
  · 감정선 처리: 감정명 직접 진술은 회차당 3회 이내. 실패 공포와 용기는 손, 시선, 댓글 창 재확인, 원고 수정 행동으로 드러낸다.
  · 금지 문체: 업계 강의체, 보고서식 요약문, 과도한 메타 밈, 사후 예고형 문장 남발 금지.
  · 금지 전개: 표절 미화, 회귀 지식 단독 해결, 경쟁자 비열한 짓밟기, 댓글/조회수 반복만으로 갈등 대체, 작중작 설명 장기 정지 금지.
```

### §A-3. 5번 세계관 설계 프롬프트용 카드 (→ 5번에서 사용)

```markdown
- 장르: 현대판타지 / 작가물 / 회귀물
- 플랫폼: 문피아 감각 + 카카오페이지 남성향 감각
- 목표 회차: 사용자 입력 없음 — 사용자 승인 필요
- 키워드: 웹소설 편집자, 실패 작가, 회귀, 연재 플랫폼, 초반 성적, 알고리즘, 생활비, 마감, 경쟁 작가, 담당 편집자, 고정 독자/댓글러
- 세계관 방향:
  · 현대 웹소설 연재 생태계를 배경으로 한다.
  · 권력/조직 축은 플랫폼 알고리즘, 편집부, 랭킹/추천 구조, 댓글/선작/조회수 반응이다.
  · 갈등은 생활비와 마감 압박, 초반 성적 실패 위험, 경쟁 작가의 소재 선점 시도, 담당 편집자와의 불신에서 발생한다.
  · 주인공의 불리함은 돈이 거의 없고, 전생 실패 기억 때문에 숫자와 반응에 집착하며, 초반 성적이 안 나오면 바로 묻히는 구조에 놓였다는 점이다.
- 절대 양보 못하는 한 끗: 표절·기만·후배 작가 짓밟기 없이, 독자의 기대를 읽고 작품 구조를 고쳐 정면승부로 성공한다.
- 피하고 싶은 것 (사용자 회피): §A-1과 동일.
- 어휘 고정 / 금지 어휘: §A-1과 동일.
- signature 톤·분위기 항목:
  · 서술자 거리: 강도윤의 제한적 3인칭 또는 밀착 3인칭을 기본으로 유지한다. 회차 내 외부 시점 전환은 1회 이내, 댓글/랭킹 화면은 독립 정보 블록처럼 길게 두지 않고 주인공의 판단 압박에 붙인다.
  · 감정 표현 수위: 감정 단어 직접 진술은 회차당 3회 이내. 실패 공포·자기혐오·용기는 원고 수정, 댓글 창 확인, 마감 시간 확인, 손/시선/호흡 반응으로 우선 표현한다.
```

### §A-4. 6번 캐릭터 설계 프롬프트용 카드 (→ 6번에서 사용)

```markdown
- 장르: 현대판타지 / 작가물 / 회귀물
- 플랫폼: 문피아 감각 + 카카오페이지 남성향 감각
- 키워드: 실패 작가의 회귀, 독자 존중, 정면승부, 담당 편집자, 경쟁 작가, 첫 고정 독자/댓글러
- 캐릭터 방향:
  · 주인공 강도윤: 34세. 전생에는 웹소설 편집자였고 작가로 전향했다가 실패했다. 회귀 후에는 전생에 망친 작품을 다시 써서 성공하려 한다.
  · 결핍: 편집자 경력 때문에 시장과 독자를 안다고 착각했고, 독자가 원하는 재미보다 자기 기준의 완성도를 앞세웠다.
  · 욕망: 전생에 망친 작품을 이번 생에는 성공시키고, 독자가 기다리고 좋아하는 이야기를 쓰는 작가로 인정받고 싶다.
  · 내부 약점: 실패 공포가 올라오면 독자 반응을 과하게 계산하고, 자기 작품을 남의 작품처럼 차갑게 재단한다.
  · 담당 편집자: 처음에는 실패한 작가를 의심하고, 강도윤도 편집자를 불신한다. 이후 서로의 실력을 인정하는 중심 관계가 된다.
  · 경쟁 작가: 소재 선점과 방해를 통해 외부 압박을 만든다. 단, 실제 작가 저격처럼 보이지 않게 설계한다.
  · 첫 고정 독자/댓글러: 초반 사소한 댓글 하나로 등장하지만, 강도윤이 독자를 숫자가 아니라 사람으로 보게 만드는 관계 축이다.
- 차별화 방향: 1번에서는 최종 확정하지 않음. “독자 존중과 작품 구조 개선으로 회귀물 치트를 대체한다”를 사용자 의지값으로 보존한다.
- 절대 양보 못하는 한 끗: 표절하지 않고, 독자를 속이지 않고, 후배 작가를 짓밟지 않는다.
- 피하고 싶은 것 (사용자 회피): §A-1과 동일.
- 어휘 고정 / 금지 어휘: §A-1과 동일.
- 사용자 톤 단서: §A-2와 동일.
- 비교 기준 메타 인물 처리 항목:
  · 대사 밀도: 갈등 장면은 대사 45~55%를 유지하되, 설명성 업계 강의가 3문단 이상 이어지면 장면 목표를 인물 선택/압박으로 되돌린다.
  · 감정선 처리: 강도윤의 변화는 “독자를 숫자로 봄→댓글 뒤의 사람을 인식함→작품을 다시 마주함” 순서로 행동을 통해 누적한다. 직접 감정명은 회차당 3회 이내.
```

