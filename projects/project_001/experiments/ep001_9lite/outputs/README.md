# 9-lite 테스트 산출물 보관용

이 폴더는 정본 `drafts/`가 아니다.

이 폴더는 9-lite 테스트 산출물 보관용이다.

`drafts/episode_001~004.md`와 별도 트랙이다.

정본 반영 전 비교·구조 보강·수동 재작성 재료로만 사용한다.

## 1~4화 테스트 결과

- 1화: 오배송/신생아 코미디 테스트
- 2화: 성장 압축/건조 회상 테스트
- 3화: 본편 현재 첫 귀신 사건 운용 테스트
- 4화: 친구 축 진입/대사 핑퐁/5화 단서 테스트

## 현재 판정

- 9-lite는 정본 대체본이 아니라 구조 보강 트랙이다.
- 돈 되는 사건 압력은 drafts 정본 트랙이 더 강하다.
- 9-lite 결과물은 수동 원고와 섞을 구조·관계·장면 후보로 사용한다.
