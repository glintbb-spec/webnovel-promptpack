# contact_channel.md

## 평시 기본선 — 감시자 경유

모든 외주·공문·요청이 감시자(차사 인턴) → 한도윤.

한도윤은 감시자 얼굴만 봐도 *또 일 떨어졌구나* 싶어 짜증.

## 변주 — 직접 방문자

특정 에피소드에서 감시자 건너뛰고 직접 방문자가 옴.

직접 방문자는 너무 자주 쓰지 않는다. 평시에는 감시자 인턴 경유가 기본선이어야 한다. 직접 방문이 잦아지면 감시자 인턴의 기능이 약해지고, 저승이 쉽게 개입할 수 있는 조직처럼 보인다. 직접 방문은 긴장감과 음모 축을 여는 변주로 제한한다.

### 발생 조건

- 감시자 권한 넘는 급
- 긴급 상황
- 메인 아크 관련 사안
- 저승 내부 책임 회피나 은폐와 연결되는 사안
- 메인 아크 단서가 드러나는 사안
- 기존 귀계 질서로 설명되지 않는 예외 상황

### 방문자 종류 (상황별)

- 저승사자
- 명부 관료
- 상급 차사
- 망자 본인
- 잡귀
- 기타

### 효과

평시 기본선을 깨뜨리는 장치로 쓰여 *긴장감·음모 축* 확장.
